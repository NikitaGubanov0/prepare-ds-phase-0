#### Основные элементы оформления
- заголовки: \# ... #######
- буллеты: - * +
- \[текст](ссылка)
- \![alt text]\(ссылка на картинку "название")
- \`inline code`  \```\n multiline code \n```
- \> цитата
- 2 пробела, \\newline или \<br/>
